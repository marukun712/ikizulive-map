# イキヅライブ聖地巡礼マップ

イキヅライブ！ LOVELIVE! BLUEBIRDの聖地情報をまとめた非公式サイトです。

# Requirements

Nix

direnv

# Install

direnvを有効化してDevShellに入ります。

```bash
direnv allow
```

依存パッケージをインストールします。

```bash
bun i
```

開発サーバーを起動します。

```bash
bun run dev
```

# Contribution

./public/data.jsonを編集することで聖地情報を編集することができます。

追加、誤りの訂正などがありましたら、お気軽にIssueやPRでご報告ください。
